# LuxeStay Hotel Website

A modern, premium fullstack hotel booking website with animations and effects.

## Features
- Responsive, animated frontend (HTML, CSS, JS)
- Premium design and smooth effects
- Node.js/Express backend
- MongoDB for hotel and booking data
- Booking form and hotel info

## Getting Started

### Backend
1. `cd server`
2. `npm install`
3. `npm start` (Runs on http://localhost:5000)

### Frontend
Open `client/index.html` in your browser.

## Project Structure
- `/client` — Frontend code (HTML, CSS, JS)
- `/server` — Backend API (Node.js, Express, MongoDB)

## Customization
- Update hotel info in the backend or connect to a real database.
- Enhance frontend with more pages, effects, or booking features.
